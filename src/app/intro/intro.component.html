Photo contest

Welcome to our photo contest! Please take your time to vote for the best photos. This website will help you to make this decision by presenting a face-off between 2 pictures several times, until it is able to tell what is your preference order for all photos.

Additional details

Merge sort

With the merge sort option, we will run a bottom up merge sort (Wikipedia reference) and you will be responsible for making each comparison. Since the number of comparisons will depend on the elements order before merging, that's why it may be necessary to run less than {{mergeSortQuantity}} comparisons. In the end, 1000 points in total will be spread linearly based on the resulting order (e.g. [0, 100, 200, 300, 400]).

Elimination

With the elimination option, we will run a single-elimination tournament (Wikipedia reference) and you will be responsible for deciding the outcome of each face-off. In the end, 1000 points in total will be given and each additional round grants double the points from the previous round (e.g. [0, 125, 125, 250, 500]).

How are the votes computed?

This is just a static page written in Angular which will provide in the end a link to a Google Forms page with the options already filled. Then you just need to login (to avoid duplicate votes) and submit. Additional validation can be made on the results to ensure that votes are valid based on the possible score distribution given by this page.

Why are there 2 different types of voting mechanisms?

Because we couldn't decide which to use!
